* PkgMega
* A PKG Installer, Homebrew Tool finder, Theme Installer, Backup Manager, and File Manager, all in one.
* Forked from PKGi, ps3 edition (as you can tell)


*Installation Instructions
1: Download PkgMega's latest release.
2: Put on a USB Flash drive or FTP Server
3: Put on the packages folder on either storage device. (If using FTP, go to dev_hdd0 first.)
4: Go to Package Manager on your PS3, select your storage device, and install!
